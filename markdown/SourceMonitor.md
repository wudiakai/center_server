# SourceMonitor

## 1.功能

### 1） lastSource

可以在系统启动时恢复上次关机时正在播放的媒体应用

支持以下策略，根据客户需求选用，可通过[config模块]()配置策略

- 不恢复（默认启动launcher）
- 每次启动固定的App
- 恢复最后播放的媒体（※可配置哪些源参与）
- *可根据需求增加策略*

### 2）Hardkey响应

支持一些Hardkey的响应，拉起相应的模块，包名可配置[config模块]()。

- Radio

- Navi   

- AVM 

- MODE循环切各个媒体（可配置哪些源参与）

  (※可配置在USB/BT未连接时，跳过相关的源)

- *可扩展*

### 3）辅助功能

- Start app
- 判断应用是否在前台
- 判断服务是否已启动
- 等
